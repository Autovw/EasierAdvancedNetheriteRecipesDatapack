Description:
Easier Advanced Netherite Recipes is a Datapack designed for the Advanced Netherite mod made by Autovw.
This Datapack adds in the old crafting recipes from Advanced Netherite 1.0.0 - 1.3.0.
Crafting recipes can be viewed here: https://github.com/Autovw/AdvancedNetherite/wiki/Classic-Crafting-Recipes
------------------------------------
Disclaimer:
This Datapack should only be downloaded from: https://github.com/Autovw/EasierAdvancedNetheriteRecipesDatapack
Do not trust any other download sources as they might be malicious.
------------------------------------
Installation:
Put the .zip file inside: 'C:\Users\{USER}\AppData\Roaming\.minecraft\saves\{WORLD}\datapacks\'
Upon joining the world trigger the /reload command and the Datapack should enable itself.
